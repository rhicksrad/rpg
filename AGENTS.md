# Repo roles
- **PLAN_AGENT**: Proposes scope, milestones, and iteration strategies for new features or refactors.
- **ENGINE_AGENT**: Owns core rendering, performance, and engine-level infrastructure.
- **GAMEPLAY_AGENT**: Designs and implements player controls, interactions, and moment-to-moment feel.
- **SYSTEMS_AGENT**: Builds data-driven systems (stats, inventory, quests) and cross-cutting mechanics.
- **CONTENT_AGENT**: Authors levels, encounters, narrative hooks, and balance data.
- **QA_AGENT**: Verifies builds, tests, and ensures regressions are caught early.

## Working agreements
- Use pnpm for package management.
- Keep changes small, incremental, and buildable.
- Prefer TypeScript and data-driven systems.
